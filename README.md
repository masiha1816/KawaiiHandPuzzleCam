# Kawaii Hand Puzzle Cam

A cute, cozy, pastel webcam puzzle game built with Python, OpenCV, MediaPipe Hands, NumPy, and optional Pygame sounds.

The app opens your webcam, tracks your hand, lets you zoom with a pinch gesture, captures a photo with a peace sign, breaks the photo into puzzle pieces, and lets you put the puzzle back together using hand-controlled pinch grabbing.

## Install

Open a terminal inside the project folder and run:

```bash
pip install opencv-python mediapipe numpy pygame
```
